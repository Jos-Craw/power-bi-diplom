# Аналитическая система визуализации данных (Дипломный проект) 📊

Репозиторий содержит практическую часть моего дипломного проекта, выполненного в Power BI. Работа охватывает полный цикл создания отчетности: от обработки сырых данных до настройки сложных интерактивных дашбордов.

## 📌 О проекте
В рамках работы была спроектирована и реализована система многомерного анализа данных для различных бизнес-сфер. Проект демонстрирует умение работать с гетерогенными данными и превращать их в инструмент для принятия решений.

**Основные кейсы в составе проекта:**
* **Банковский сектор:** Анализ сети банкоматов и финансовых потоков.
* **Медицина:** Визуализация показателей медицинского центра.
* **Филиальная сеть:** Сравнительный анализ эффективности подразделений.
* **Ресурсная база:** Учет и аналитика (драгметаллы и сырье).

## 🛠 Технологический стек
* **Инструмент:** Microsoft Power BI
* **Моделирование данных:** Схема «Звезда», настройка связей (One-to-Many).
* **Вычисления:** DAX (расчет мер, KPI, динамические фильтры).
* **ETL:** Power Query (очистка и трансформация данных).

## 📂 Описание файлов
* `Диплом Новоженцев.docx` — полная пояснительная записка с методологией.
* `atm.pbix`, `atm-country.pbix` — отчеты по банковской инфраструктуре.
* `финансы.pbix`, `филиалы.pbix` — финансовая налитика.
* `мед.pbix` — аналитика для медицинской сферы.
